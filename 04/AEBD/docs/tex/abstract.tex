
\begin{abstract}
Este relatório serve como descrição dos procedimentos realizados para o trabalho prático de Administração e Exploração de Bases de Dados. Este projeto tem o objetivo de, utilizando todos os conhecimentos adquiridos na componente prática e teórica da UC de Administração e Exploração de Base de Dados, construir um pequeno monitor de BD que apresente de forma simples os principais parâmetros de avaliação de performance de uma BD Oracle.
\end{abstract}
